# Phase 6: 온라인 결제 (추후 개발)

## 이 Phase의 목표

리드 → 데모 → 계약까지의 프로세스가 안정화된 후, 온라인 견적/결제 시스템 도입.

> 이 Phase는 Phase 5까지 운영하면서 실제 니즈가 확인된 후 개발합니다.
> 현재는 방향성만 기록합니다.

---

## 예상 기능

### 1. 온라인 견적 시스템

```
admin에서 리드 선택 → 견적서 자동 생성:
- 제품 정보 + 가격 (제품 테이블에서)
- 결제 조건 (일시불/할부/렌탈/캐피탈)
- 추가 옵션 (설치비, 교육비, 유지보수)
- 견적 유효기간
- PDF 생성 + 이메일/카카오 발송
```

### 2. 제품별 결제 방식

```
고가 장비 (TORR RF 등):
- 계약금 + 잔금 (은행 이체)
- 의료기기 캐피탈 (리스)
- 할부 (12/24/36개월)
→ 온라인 결제보다 은행 이체 + 계약서 중심

소모품:
- 정기 주문 (월/분기)
- 온라인 결제 (카드)
- 대량 할인 자동 적용
→ 온라인 결제 적합

중가 장비:
- 케이스 바이 케이스
```

### 3. 기술적 방향

```
- PG 연동: 토스페이먼츠 or 나이스페이먼츠
- 견적서 PDF: @react-pdf/renderer
- 전자 계약서: 카카오 전자서명 or 모두사인
- 정기 결제: 소모품 정기 배송용
```

---

## 개발 시점

Phase 5 런칭 후 3개월 운영 데이터를 보고 결정:
- 월 계약 건수가 5건+ 안정적이면 → 견적 자동화 우선
- 소모품 반복 주문이 늘면 → 온라인 결제 우선
- 둘 다 아니면 → 수동 운영 유지
